# ESP32-S3 开发工具链

## 一、背景

ESP32-S3 是乐鑫科技 (Espressif) 推出的低功耗 WiFi + Bluetooth LE (BLE) SoC，集成了 Xtensa® 32-bit LX7 双核处理器，支持 AI 加速向量指令，非常适合作为机器人的实时控制器。

Mini-Bot 项目使用 ESP32-S3 作为底层控制器，负责电机 PWM 控制、IMU 数据采集、舵机控制等实时任务。

## 二、目的

- **电机控制**：4 路 PWM 输出控制 TB6612FNG 驱动
- **IMU 采集**：通过 I2C 读取 MPU6050 数据
- **舵机控制**：PCA9685 或 ESP32 内置 PWM 控制机械手舵机
- **通信桥接**：与树莓派 5 串口通信，接收 ROS2 指令
- **WiFi/BLE**：可选无线调试和固件更新

## 三、官方地址

## 四、实用教程

### 4.0 检查安装状态 (macOS)

```bash
# 检查 PlatformIO 是否已安装
pio --version

# 如果命令找不到，添加 PATH
export PATH="$HOME/Library/Python/3.9/bin:$PATH"
```

### 4.1 安装 ESP-IDF (官方 SDK)

```bash
# 方法1: 一键安装脚本（推荐）
mkdir -p ~/esp
cd ~/esp
git clone --recursive https://github.com/espressif/esp-idf.git
cd esp-idf
./install.sh esp32s3

# 激活环境
. ./export.sh

# 永久激活（添加到 .bashrc）
echo 'alias get_idf="source ~/esp/esp-idf/export.sh"' >> ~/.bashrc
```

### 4.2 PlatformIO (更简单的方案)

PlatformIO 是一个跨平台的嵌入式开发平台，支持 ESP32、Arduino 等多种芯片，配置简单。

```bash
# 安装 PlatformIO
pip install platformio

# 或使用 VSCode 插件
code --install-extension platformio.platformio-ide

# 创建项目
mkdir ~/mini_bot_firmware && cd ~/mini_bot_firmware
pio project init --board esp32dev --framework arduino

# platformio.ini 配置
cat > platformio.ini << 'EOF'
[env:esp32-s3-devkitc-1]
platform = espressif32
board = esp32-s3-devkitc-1
framework = arduino
monitor_speed = 115200
upload_speed = 921600

[env:esp32dev]
platform = espressif32
board = esp32dev
framework = arduino
EOF
```

### 4.3 编写 ESP32-S3 电机控制代码

```cpp
// src/main.cpp
#include <Arduino.h>

// 引脚定义
#define MOTOR_A_PWM 10
#define MOTOR_A_DIR1 11
#define MOTOR_A_DIR2 12
#define MOTOR_B_PWM 13
#define MOTOR_B_DIR1 14
#define MOTOR_B_DIR2 15

#define I2C_SDA 1
#define I2C_SCL 2

// 串口命令解析
String cmd = "";
int targetLeftSpeed = 0;
int targetRightSpeed = 0;

void setMotorSpeed(int pwm, int dir1, int dir2, int speed) {
    if (speed > 0) {
        digitalWrite(dir1, HIGH);
        digitalWrite(dir2, LOW);
    } else if (speed < 0) {
        digitalWrite(dir1, LOW);
        digitalWrite(dir2, HIGH);
        speed = -speed;
    } else {
        digitalWrite(dir1, LOW);
        digitalWrite(dir2, LOW);
    }
    ledcWrite(0, speed);  // PWM channel 0
}

void parseCommand(String cmd) {
    if (cmd.startsWith("L") && cmd.indexOf("R") > 0) {
        int sep = cmd.indexOf("R");
        String leftStr = cmd.substring(1, sep);
        String rightStr = cmd.substring(sep + 1);
        targetLeftSpeed = leftStr.toInt();
        targetRightSpeed = rightStr.toInt();
    }
}

void setup() {
    Serial.begin(115200);
    
    // 配置电机引脚
    pinMode(MOTOR_A_DIR1, OUTPUT);
    pinMode(MOTOR_A_DIR2, OUTPUT);
    pinMode(MOTOR_B_DIR1, OUTPUT);
    pinMode(MOTOR_B_DIR2, OUTPUT);
    
    // 配置 PWM
    ledcAttach(MOTOR_A_PWM, 1000, 8);  // 1kHz, 8-bit
    ledcAttach(MOTOR_B_PWM, 1000, 8);
    
    Serial.println("Mini-Bot ESP32-S3 Ready");
}

void loop() {
    // 接收串口命令
    while (Serial.available()) {
        char c = Serial.read();
        if (c == '\n') {
            parseCommand(cmd);
            cmd = "";
        } else {
            cmd += c;
        }
    }
    
    // 设置电机速度
    setMotorSpeed(MOTOR_A_PWM, MOTOR_A_DIR1, MOTOR_A_DIR2, targetLeftSpeed);
    setMotorSpeed(MOTOR_B_PWM, MOTOR_B_DIR1, MOTOR_B_DIR2, targetRightSpeed);
    
    delay(10);
}
```

### 4.4 上传与调试

```bash

cd ~/mini_bot_firmware
pio run --target upload
pio device monitor

# 使用 ESP-IDF

# 设置串口权限（Linux）

# 重新登录后生效
```

## 五、工具链对比

| 工具 | 优点 | 缺点 |
|------|------|------|
| **PlatformIO** | 配置简单，VSCode 插件优秀 | 定制化有限 |
| ESP-IDF | 完整控制，最新特性 | 配置复杂 |
| Arduino IDE | 上手最简单 | 功能有限 |

**推荐**：Mini-Bot 项目使用 **PlatformIO**，配置简单且与 VSCode 集成良好。

# ROS2 Humble

## 一、背景

ROS (Robot Operating System) 是由 Willow Garage 公司于 2007 年发起的一个开源机器人软件框架，现由 Open Robotics 维护。ROS2 是 ROS 的下一代版本，于 2022 年发布"Humble Hawksbill" LTS 版本。

ROS 不是传统意义上的操作系统，而是一个元操作系统（meta-operating system），提供硬件抽象、设备驱动、库函数、工具集等功能，让机器人软件开发变得更简单。

## 二、目的

- **机器人控制**：提供标准化的接口来控制电机、读取传感器
- **通信中间件**：基于 DDS (Data Distribution Service) 的实时通信
- **导航定位**：SLAM、路径规划、自动导航
- **工具生态**：RViz 可视化、rqt 调试工具、tf 坐标变换等

## 四、实用教程

### 4.1 安装 ROS2 Humble (Ubuntu 22.04)

```bash
# 1. 设置 locale
sudo locale-gen en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8

# 2. 添加 ROS2 GPG 和 源
sudo apt install -y curl gnupg lsb-release
sudo curl -ssl https://raw.githubusercontent.com/ros/rosdistro/master/ros.key | sudo gpg --dearmor -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(source /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# 3. 安装
sudo apt update
sudo apt install -y ros-humble-ros-base
sudo apt install -y python3-colcon-common-extensions

# 4. 安装额外包（推荐）
sudo apt install -y ros-humble-robot-localization ros-humble-robot-navigation
sudo apt install -y ros-humble-teleop-twist-keyboard
```

### 4.2 创建工作空间

```bash
# 创建工作空间
mkdir -p ~/mini_bot_ws/src
cd ~/mini_bot_ws

# 初始化
source /opt/ros/humble/setup.bash
colcon build
source install/setup.bash

# 验证
ros2 pkg list | grep mini_bot
```

### 4.3 创建第一个 ROS2 包

```bash
cd ~/mini_bot_ws/src
ros2 pkg create mini_bot_control --build-type ament_python
cd mini_bot_control/mini_bot_control

# 创建节点 (chassis_node.py)
cat > chassis_node.py << 'EOF'
import rclpy
from rclpy.node import Node
from geometry_msgs.msg import Twist

class ChassisNode(Node):
    def __init__(self):
        super().__init__('chassis_node')
        self.cmd_pub = self.create_publisher(Twist, '/cmd_vel', 10)
        self.timer = self.create_timer(0.1, self.timer_callback)
        
    def timer_callback(self):
        msg = Twist()
        msg.linear.x = 0.1
        self.cmd_pub.publish(msg)

def main(args=None):
    rclpy.init(args=args)
    node = ChassisNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
EOF

# 编译
cd ~/mini_bot_ws
colcon build --packages-select mini_bot_control
source install/setup.bash

# 运行
ros2 run mini_bot_control chassis_node
```

### 4.4 常用命令

```bash
# 列出所有包
ros2 pkg list

# 创建 Launch 文件
ros2 launch mini_bot_bringup bringup.launch.py

# 查看话题
ros2 topic list
ros2 topic echo /cmd_vel

# 启动键盘控制
ros2 run teleop_twist_keyboard teleop_twist_keyboard
```
